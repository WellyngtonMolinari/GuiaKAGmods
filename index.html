Guia de Criação de Mods para King Arthur's Gold

1. Introdução ao Mod

Neste guia, vamos aprender como criar um mod básico para o jogo King Arthur's Gold, chamado "Cavaleiro Sanguinário". Este mod ajustará os tempos de ataque da classe cavaleiro, tornando-a mais ágil e feroz. Além disso iremos adicionar o efeito vermelho no slash do sprite do cavaleiro! Primeiramente você precisa de um editor de código, eu estarei usando neste guia o Visual Studio Code para modificar os scripts do jogo.Se quiser, pode baixar o arquivo completo do Mod aqui!
Lembre-se de que por padrão, o jogo vai carregar todos os arquivos originais que estão na pasta "Base", a não ser que eles forem modificados e colocados na pasta "Mods" (que é o que iremos fazer neste guia).

2. Localizar Arquivos do Jogo

Localize os arquivos responsáveis pela lógica do cavaleiro no jogo:

  • Abra a pasta raiz de instalação do King Arthur's Gold.
    Geralmente em "C:\SteamLibrary\steamapps\common\King Arthur's Gold"
  • Navegue até a pasta "Base", onde estão os scripts do jogo.
    • "C:\SteamLibrary\steamapps\common\King Arthur's Gold\Base"
  • Encontre o arquivo específico que controla os scripts da classe cavaleiro, que é "KnightCommon.as"
     "C:\SteamLibrary\steamapps\common\King Arthur's Gold\Base\Entities\Characters\Knight"
  • Anote o endereço completo da pasta e copie o arquivo "KnightCommon.as".

3. Fazer Modificações

Agora que sabemos onde estão os arquivos, vamos começar a fazer as modificações necessárias:

  • Volte à pasta raiz do jogo.
  • Crie uma nova pasta para o seu mod dentro da pasta "Mods", nomeie-a como "CavaleiroSanguinario".
  • Agora crie mais pastas, nessa ordem, uma dentro da outra: "Entities", "Characters", e "Knight". Cole o arquivo "KnightCommon.as" dentro dessa última pasta, mantendo o mesmo nome e estrutura de diretórios. Dessa forma: "\Mods\CavaleiroSanguinario\Entities\Characters\Knight"
  • Abra o arquivo copiado e estude a lógica do script.
  • Procure pela variável que controla o tempo entre os ataques do cavaleiro e faça as modificações desejadas para ajustá-la. Mas tome cuidado, pois você pode quebrar o script e o jogo não vai funcionar! Eu diminuí pela metade praticamente todas as variáveis.
  • Salve as alterações no arquivo.
  • Você também pode modificar o sprite do cavaleiro e adicionar efeitos no slash dele, seguindo essa mesma lógica de modificação de arquivos usada até aqui.

4. Testar o Mod

Com as modificações feitas, é hora de testar o nosso mod:

  • Volte à pasta raiz do jogo.
  • Encontre o arquivo mods.cfg.
  • Adicione o nome da pasta do mod "CavaleiroSanguinario" na primeira linha após os comentários deste arquivo e salve-o.
  • Execute o jogo usando o arquivo runlocalhost para iniciar com o mod carregado.
  • Parabéns, você terminou seu primeiro mod!
  • Teste o jogo e observe as mudanças no comportamento do cavaleiro.
