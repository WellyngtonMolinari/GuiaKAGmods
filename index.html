Guia de Criação de Mods para King Arthur's Gold

1. Introdução ao Mod

Neste guia, vamos aprender como criar um mod básico para o jogo King Arthur's Gold, chamado "Cavaleiro Sanguinário". Este mod ajustará os tempos de ataque da classe cavaleiro, tornando-a mais ágil e feroz.

2. Localizar Arquivos do Jogo

O primeiro passo é localizar os arquivos responsáveis pela lógica do cavaleiro no jogo:

  • Abra a pasta de instalação do King Arthur's Gold.
  • Navegue até a pasta onde estão os scripts do jogo.
  • Encontre o arquivo específico que controla os scripts da classe cavaleiro, geralmente nomeado algo como knight.as.
  • Anote o endereço completo da pasta e o copie o arquivo completo.

3. Fazer Modificações

Agora que sabemos onde estão os arquivos, vamos começar a fazer as modificações necessárias:

  • Volte à pasta raiz do jogo.
  • Crie uma nova pasta para o seu mod dentro da pasta "Mods", nomeie-a como CavaleiroSanguinario.
  • Copie o arquivo knight.as para a pasta do mod, mantendo o mesmo nome e estrutura de diretório.
  • Abra o arquivo copiado e estude a lógica do script.
  • Procure pela variável que controla o tempo entre os ataques do cavaleiro e faça as modificações desejadas para ajustá-la.
  • Salve as alterações no arquivo.

4. Testar o Mod

Com as modificações feitas, é hora de testar o nosso mod:

  • Volte à pasta raiz do jogo.
  • Encontre o arquivo mods.cfg.
  • Adicione o nome da pasta do mod CavaleiroSanguinario na primeira linha deste arquivo e salve-o.
  • Execute o jogo usando o arquivo runlocalhost para iniciar com o mod carregado.
  • Teste o jogo e observe as mudanças no comportamento do cavaleiro.
